# SRS Summary - Key Functional Requirements

## User Registration & Authentication
- FR 1: Register using institutional email
- FR 2: Anonymous profile option
- FR 3: Role assignment (Student, Counsellor, Admin)
- FR 4: Secure login with hashed passwords

## Mood Check-In
- FR 8: Log daily mood (1-5 scale)
- FR 9: Optional journal notes
- FR 10: View mood history chart
- FR 11: Crisis alert for 3+ consecutive low moods

## Self-Help Resources
- FR 14: Categorized resource library
- FR 15: Search by keyword
- FR 17: Crisis helpline on all pages

## Appointment Booking
- FR 21: View available slots
- FR 22: Book appointment with email notification
- FR 24: Cancel/reschedule 24 hours before

## Counsellor Dashboard
- FR 28: Appointment calendar view
- FR 29: Confidential session notes
- FR 30: Set availability
